# AI-Based Resume Screening System

This project ranks resumes based on similarity to a job description using Natural Language Processing.

## Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity

## How It Works
1. Reads job description
2. Reads multiple resumes
3. Converts text to numerical vectors using TF-IDF
4. Calculates similarity scores
5. Ranks resumes based on relevance
